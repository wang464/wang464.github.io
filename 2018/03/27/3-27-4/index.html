贸易战前夜 平民如何应对？——美国向中国商品征收50亿美元关税

1

美帝剑指中国

继美国向对进口钢铁征收25%、对进口铝征收10%的关税后，却又指出加拿大、墨西哥、澳大利亚、阿根廷、巴西、南韩、欧盟的关税将被暂时免除。主要出口国都得到了例外，还有哪个国家不是例外？答案显而易见，中国、日本、台湾地区、俄罗斯、土耳其。


这次美帝征收关税的靴子落地，向中国1500种商品征收500亿美元关税，剑指中国，中美间的贸易战看来已经不可避免。毕竟美国去年与中国存在3370亿美元的贸易逆差，比2016年还增加了9%。
这说明刘副总理在两会期间去美国与Mnuchin、Lighthizer、Cohen的商务磋商并不乐观，所以回来以后立刻积极准备迎战。

1.1

谁的赢面大？

川普在推特上说打贸易战美国必胜，因为美国已经是在贸易中吃亏的一方。但是

  • 中国从美国获得贸易逆差，然后又用外汇储备买了美国国债，这个事川普没说；
  • 美国居民获得了廉价的商品，这个事川普没说；
  • 中国不能失去美国这个大客户，但是美国也无法失去中国这样一个大供应商。 美帝的困境：向中国征收关税->美国物价上涨->居民生活水平下降->选民的报复
    中国的困境：向美国农产品和高科技产品征收报复性关税->中国物价上涨->货币供应量减少->企业出口减少->经营态势恶化

    1.2

中国做了哪些反制措施？

  1. 我们看见证监会已经做了充足甚至是最严格的准备，一方面劝退拟IPO的企业，一方面迎接中概股的回归。
  2. 另一方面，中方将向WTO提出申诉。
  3. 但是估计中方打贸易战的决心不足，毕竟2017年中美间的贸易逆差达到了3370亿美元；如果从这个角度看，500亿美元是一个令人恶心但是还没到伤筋动骨的事，还不值得掀桌。
  4. 2017年中国采购了美国380亿美元的服务（含软件），我猜这部分可能会有一个有限程度的反击。
  5. 中国握有大量美国国债，但是抛售美国国债等于让中国外汇资产缩水，所以这点也难以反击。
  6. 反击就是必须的，虽然表面上看起来中美间存在巨额的贸易逆差，但是中美两国的政体不一样。中国政体能够承受长期经济的不景气，能够长期团结在党中央周围；而川普不行，经济一旦下行、物价一旦上涨25%，那美国人民就会用选票说话。

    2

神仙打架 老三遭殃

过去几次美国跟日本的经济争端也好、广场协议也好，并不适用于今天的中美关系，因为中国是独立自主的拥核国家，而非日本这种有美国驻军的不自主国家，美方永远能够把日方逼回谈判桌并接受苛刻的条款。日方是不想谈，但是怕局势恶化；中国是愿意谈，但是不惧贸易战。

2017年，美国GDP 19.362万亿美元，中国GDP82万亿人民币。按现行汇率计算，中国的GDP已经达到了美帝的68%，已经具备了挑战美国霸主地位的资格，所以中美双方必定会发生冲突和校力。历史上的两个挑战者，分别是苏联和日本，结果大家也都知道。

所以不要抱着忍让求存的幻想，不可能。全球老大不可能放弃打压竞争者的手段的，就算WTO合规，美国仍然可以发动301条款。


USTR，贸易代表办公室，早在1992年就曾经以知识产权保护为由，计划对中国输往美国的106种商品加征100%关税；最后关头中美双方签署了知识产权备忘录。

301条款，是指美国1974年《贸易法》中规定，如果贸易代表办公室（USTR，现在的负责人是Lighthizer，刘副总理此行去美国见了他，里根时代的老资历）在经过调查之后，认为其他国家侵犯了美国权利，或者这个国家的法律法规对美国商业造成了限制或负担，可以采取报复措施。
中美的产业格局已经存在了重叠，中国在努力往智能制造的方向上提升，而美国也在努力推进制造业回流。

但是仅就加工制造来说，目前中国世界工厂的地位还无法撼动，因为制造业的体系包括物流、配套、能源、原料、规模效应、高素质的工人，这些中国有极大的优势，不是一时就能替代的。

美国损失的主要是对华出口的农产品、高科技产品、服务，但是中国一时半会找不到其他卖家，同样美国也找不到同样规模的买家，这点也是互相威慑的。

所以，贸易战可能只是幌子，只是漫天要价、就地还钱的手段，在商场上磨练了一辈子的川普大帝一定深谙此道；但是美国联邦政府并不是一个团结在川普周围的组织，每个人都各怀心思；一个不慎，贸易战的规模和烈度就会失控；毕竟共和党中期国会选举失利，民主党正随时准备着扯共和党的后腿呢，在政策执行过程中存心添堵，还是非常容易的。

所以，这场仗打不起来，但是中美之间，挑战者与世界警察之间，终有一战。

3

平民百姓 未雨绸缪

中美贸易逆差3370亿美元，所以中国对内能够增发2.15万亿的货币，最直观的感受是我们手头都很宽松了，中国变得有钱了，大家可以全球旅游了。

3.1

影响极大

然而这场国家层面的贸易战即使只持续一两年，对于我们这些平民的生活都有非常大的影响。

  • 出口企业没有海外订单，他们的员工生活水平下滑；
  • 中国的外汇收入减少，所以对口的国内人民币供应量也会减少，钱会变紧；
  • 出口企业配套供应链上的企业也会减产，因为经济是会传导的；
  • 企业不景气，员工收入减少将导致国内消费下滑，消费更多转向网购、游戏和视频。

    3.2

稳健应对

  • 在近两年里，千万不要轻易跳槽，一定要看准了再跳；
  • 千万不要贸然给自己加上几倍杠杆购买房产或者股票，任何人的收入都可能发生变化；
  • 买房的月供一定不能高于家庭收入的安全线，否则压力会非常大，就算急用钱，卖房周期也会在半年以上；
    总的来说，近两年里，只要我们作为平民百姓能够保持正常的现金流，只要没给自己加上动辄爆仓的杠杆，只要月供没有超过收入安全线，那就能稳稳的熬过去这两年。

再过两年又到了美国大选年，中美的贸易战不会一直大战下去，最多打到2020年初。
